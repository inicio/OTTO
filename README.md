OpenTag Toronto, or OTTO, is an autonomous graffiti machine. He traverses Toronto, tagging various buildings and landmarks. The comments he leaves aren't his, though. He's merely a vessel through which consciousness spreads.
========================================================

### Hardware

* [IOIO](http://github.com/ytai/ioio/wiki)
* Google Nexus One
* Actuators (7, printed on a Stratasys)
* Servos (7)
* 7.4V Dual Cell Battery
* Metal Frame

### Application

* Android SDK 4
* [Twitter4J](http://twitter4J.org)
